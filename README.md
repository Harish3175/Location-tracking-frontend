# 📍 Location Tracking Frontend (React)

## 🚀 Description
This is the frontend for the Location Tracking System. It provides a user interface to authenticate users, manage locations, view history, and interact with backend APIs using a secure and role-based system.

---

## 🛠️ Tech Stack
- React.js
- Axios
- React Router
- Tailwind CSS

---

## ✨ Features
- User Authentication (Login / Signup)
- Protected Routes (Private Routing)
- Role-Based UI (Admin / User)
- Location tracking interface
- View history of activities
- Record management
- Responsive design

---

## 🔐 Authentication

- JWT token stored after login  
- Token sent with API requests  
- Protected routes using custom components  
- Role-based UI rendering  

---

## 📂 Pages

- Login Page  
- Register Page  
- Dashboard  
- Location Tracking Page  
- History Page  
- Records Page  
- Admin Panel  

---

## 🔄 Routing

- Implemented using React Router  
- Protected routes for authenticated users  
- Role-based route access
