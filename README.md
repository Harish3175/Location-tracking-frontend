📍 Location Tracking - Frontend

This is the frontend application for the Location Tracking system. It provides a user-friendly interface to view and interact with real-time location data.

🚀 Features

- 📍 Display location data on UI
- 🔄 Real-time updates
- 🎨 Clean and responsive design
- 🔗 Integration with backend APIs

🛠️ Technologies Used

- React.js
- HTML
- CSS
- JavaScript
